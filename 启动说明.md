## 配置 LLM key（跑出真正切片的最后一步）
两种方式任选：
1. **设置页（推荐）**：打开 http://localhost:3000 → 设置 → 填 API key（默认 provider = 通义千问 DashScope）。
   保存后写入 `data/settings.json`。
2. **环境变量**：编辑 `.env`，取消 `API_DASHSCOPE_API_KEY=` 注释并填 key，重启后端。

配好后重新上传视频，即可产出真正的切片。
